# /handoff-to-claude - Claude Code に実装を依頼

PM（Cursor）から Impl（Claude Code）へタスクをハンドオフします。

## ハンドオフ内容

以下の情報を Claude Code に伝えます:

### 1. 依頼タスク
Plans.md から `cc:TODO` または `pm:依頼中` のタスクを特定。

### 2. 完了条件
- 何ができれば完了か
- テスト方法・確認方法

### 3. 制約・注意点
- 技術的制約
- 既存コードとの整合性
- 避けるべきこと

## ハンドオフメッセージ例

```
【Claude Code へ】

依頼タスク: index.html 基本構造作成

完了条件:
- HTML5 テンプレートが作成されている
- Canvas 要素が配置されている
- レスポンシブ対応のスタイルが適用されている

制約:
- 外部ファイル/CDN は使わない
- index.html 一つで完結させる

確認方法:
- python -m http.server 8000 で起動
- ブラウザで表示確認
```

---

Plans.md を確認し、依頼するタスクを特定してから
上記フォーマットでハンドオフメッセージを作成してください。
