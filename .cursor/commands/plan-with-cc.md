# /plan-with-cc - Claude Code と計画を立てる

新しい機能やタスクの計画を Claude Code と一緒に立てます。

## 使い方

1. やりたいことを伝える（例: 「プレイヤーの移動を実装したい」）
2. Claude Code が実装方針を提案
3. 方針を承認 → Plans.md にタスク追加
4. `/handoff-to-claude` で実装依頼

## 計画時のポイント

- **ゴールを明確に**: 何ができれば完了か
- **制約を伝える**: 技術的制約、時間制約
- **優先度を決める**: 必須 vs あれば良い

---

やりたいことを具体的に伝えてください。
Claude Code が実装方針を提案します。
