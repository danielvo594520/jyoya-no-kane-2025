# /start-session - セッション開始

PM（Cursor）としてセッションを開始します。

## 実行内容

1. **Plans.md を読み込み**、現在のタスク状況を把握
2. **AGENTS.md を確認**し、プロジェクト概要を理解
3. **次のアクションを提案**

## 使い方

このコマンドを実行後、以下のいずれかを選択:

- **新しいタスクを追加したい** → タスク内容を伝える
- **Claude Code に実装を依頼したい** → `/handoff-to-claude`
- **Claude Code の作業をレビューしたい** → `/review-cc-work`

---

まず Plans.md と AGENTS.md を読んで、現在の状況を確認してください。
