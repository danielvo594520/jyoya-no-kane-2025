# CLAUDE.md - Claude Code 設定

## 🎮 プロジェクト: 除夜の鐘シューティング 2025

宇宙空間で煩悩を打ち払うシューティングゲーム。

## 📋 実装ルール

### コーディング規約
- **言語**: JavaScript (ES6+)
- **スタイル**: セミコロンあり、シングルクォート
- **インデント**: 2スペース
- **命名**: camelCase（変数・関数）、PascalCase（クラス）

### ファイル構成
- `index.html` 一つで完結させる（CSS/JS インライン）
- 外部依存なし（CDN も使わない）
- レスポンシブ対応必須

### Canvas 描画
- requestAnimationFrame でゲームループ
- デバイスピクセル比対応
- タッチ/マウス両対応

## 🎯 ゲーム仕様

### 操作
- 移動: ← → / A D / マウス / タッチ
- 攻撃: スペース / クリック / タップ

### 煩悩の種類
| 種類 | 色 | 特徴 |
|------|-----|------|
| 怒 | 赤 | 高速落下 |
| 欲 | 紫 | サイズ大 |
| 妬 | 緑 | 追尾 |
| 惰 | 黄 | ゆらゆら |
| 傲 | 水色 | 通常 |

### フェーズ
1. 第一の試練 (0〜35回): 単発攻撃
2. 第二の試練 (36〜71回): 拡散攻撃
3. 最終試練 (72〜107回): 弾幕モード

## 🔧 開発コマンド

```bash
# ローカルで確認（Python）
python -m http.server 8000

# ローカルで確認（Node.js）
npx serve .
```
