# 🔔 除夜の鐘シューティング - 宇宙編

宇宙空間を舞台に、お坊さんがハンマーを投げて除夜の鐘を打つシューティングゲーム。

## 🎮 遊び方

- **移動**: ← → キー / A D キー / マウス移動 / タッチ移動
- **攻撃**: スペースキー / クリック / タップ
- **目標**: 108回鐘を打って煩悩を払う

## 👹 煩悩の種類

| 煩悩 | 特徴 |
|------|------|
| 怒（赤） | 高速で落下 |
| 欲（紫） | サイズが大きい |
| 妬（緑） | プレイヤーを追尾 |
| 惰（黄） | ゆらゆら揺れる |
| 傲（水色） | 通常タイプ |

## 📈 フェーズ

1. **第一の試練** (0〜35回): 単発攻撃
2. **第二の試練** (36〜71回): 拡散攻撃追加
3. **最終試練** (72〜107回): 弾幕モード

## 🚀 デプロイ

GitHub Pagesで公開する場合:

1. このリポジトリをフォーク or クローン
2. Settings → Pages → Source を `main` ブランチに設定
3. 数分後に `https://[username].github.io/[repo-name]/` でアクセス可能
