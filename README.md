# 🔔 除夜の鐘シューティング - 宇宙編

宇宙空間を舞台に、お坊さんがハンマーを投げて除夜の鐘を打つシューティングゲーム。

## 🎮 遊び方

- **移動**: ← → キー / A D キー / マウス移動 / タッチ移動
- **攻撃**: スペースキー / クリック / タップ
- **目標**: 108体の煩悩を倒し、最後にボス「煩悩大鐘」を撃破！

## 👹 煩悩の種類

| 煩悩 | 色 | 特徴 |
|------|-----|------|
| 傲 | 水色 | 通常タイプ |
| 怒 | 赤 | 高速で落下、左右にブレる |
| 欲 | 紫 | サイズが大きい |
| 妬 | 緑 | プレイヤーを強力に追尾 |
| 惰 | 黄 | 大きくゆらゆら揺れる |

## 📈 ゲームフロー

### フェーズ1〜3: 煩悩退治

| フェーズ | 範囲 | 攻撃パターン |
|----------|------|--------------|
| 第一の試練 | 0〜35回 | 単発攻撃 |
| 第二の試練 | 36〜71回 | 3方向拡散攻撃 |
| 最終試練 | 72〜107回 | 5方向弾幕モード |

### ボス戦: 煩悩大鐘

108体の煩悩を倒すと、巨大な鐘のボスが出現！

| 項目 | 内容 |
|------|------|
| HP | 108 |
| 移動 | ランダムに左右移動 + 突然のダッシュ |
| 攻撃1 | 煩悩ビーム（縦方向、警告後に発射） |
| 攻撃2 | 鐘から煩悩を放出 |

## 🎯 攻略のコツ

- 煩悩ビームは**警告が出たらすぐ横に逃げる**
- ボス戦中の煩悩は倒してもカウントされないが、邪魔なので適度に処理
- ボスは**ランダムにダッシュ**するので、ハンマーを連射して当てる

## 🚀 デプロイ

### GitHub Pages

1. このリポジトリをフォーク or クローン
2. Settings → Pages → Source を `main` ブランチに設定
3. 数分後に `https://[username].github.io/[repo-name]/` でアクセス可能

### ローカルで確認

```bash
# Python
python -m http.server 8000

# Node.js
npx serve .
```

http://localhost:8000 でプレイ可能
